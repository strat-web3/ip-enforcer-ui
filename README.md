# IP Enforcer UI

Protect your artwork in the most efficient way.

## Install

```bash
pnpm i
```

## Run

Create a `.env` file:

```bash
cp .env.template .env
```

Add your own keys in the `.env` file:

```env
NEXT_PUBLIC_PROJECT_ID=your_reown_project_id  # Get yours at https://cloud.reown.com/
```

Then start the development server:

```bash
pnpm dev
```
